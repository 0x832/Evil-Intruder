# Evil Intruder 

Evil Intruder es una herramienta diseñada para realizar diversas operaciones relacionadas con redes 
inalámbricas, incluyendo ataques de seguridad. Esta herramienta está destinada a ser
utilizada por personas con conocimientos avanzados en redes 
# Funciones Principales

## Requisitos 
- Linux (preferiblemente una distribución basada en Debian)
- Tarjeta de red inalámbrica compatible con modo monitor
- Herramientas como Aircrack-ng, Wireshark, Cowpatty, John the Ripper

    - ### instalación de los requisitos
  ```bash
      sudo apt-get install aircrack-ng
      sudo apt-get install wireshark
      sudo apt-get install cowpatty
      sudo apt-get install john

- Privilegios de superusuario (root)

## Instalación

- Sigue estos pasos para instalar evil-intruder en tu sistema:
   ```bash
   git clone https://github.com/0x832/evil-intruder.git
